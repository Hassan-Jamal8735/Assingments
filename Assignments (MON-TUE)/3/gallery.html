HTML5 introduces the <figure> and <figcaption>

<figure></figure>

elements to enhance the association between images and their captions. The <figure> element is used to contain images along with their captions, allowing for a clear connection between the two. Multiple images can be placed inside a <figure> element, as long as they share the same caption.

<figcaption></figcaption>

The <figcaption> element, on the other hand, is employed to provide a caption for an image within the <figure> element. This addition in HTML5 allows web page authors to create a meaningful association between an image and its corresponding caption. Before the introduction of these elements, there was no standardized way to link an image with its caption.

It's worth noting that older browsers that do not support HTML5 elements will simply ignore these new elements and display their content as normal.
